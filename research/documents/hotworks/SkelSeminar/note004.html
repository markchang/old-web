Notes:

    Types of plug-ins:
    Color Picker: when user request a custom color
    Import: imports an image into a new window
    Export: exports the image window into a user-defined file format
    Extension: session-start/session-end features (initialization of devices)
    Filter: filter an existing image
    Format: reading/writing additional formats
    Parser: manipulation of images between different image file formats
    Selection: modifies which pixels are selected (fade/feathering)
