research::

current

My current research is devoted to variable precision analysis. In particular, researching tool development in order to aid designers in performing precision analysis on algorithmic code in order to efficiently make design-time tradeoff decisions.

We envision a tool that would:

  1. Allow the user to specify bit-widths at various input stages and intermediate stages
  2. Propagate any precision information automatically throughout the system
  3. Aide the user in performing simulations to deduce bit widths, perform error analysis, and to quantify precision tradeoffs
  4. Perform data-analysis on representative data samples to hint to the user possible points of improvement
  5. Suggest to the user possible contraint opportunities that maximize performance gain (either area, power, or speed)

This tool is being developed in C++ and Java. As this work falls under the MATCH project, the front-end of the tool utilizes MATLAB as the source language. Utilizing the MATCH compiler, the tool compiles to an intermediate representation that is further parsed with a Java-based tool.

The current state of the tool is as follows:

  • Performs MATLAB to Intermediate representation
  • Allows for user-supplied variable and operator constraints
  • Performs constraint propagation
  • Generates MATLAB code for simulation
  • Provides routines to simulate truncation and overflow errors in the MATLAB environment
  • Provides routines to gather precision information from sample data sets

This tool is currently being actively worked upon.
